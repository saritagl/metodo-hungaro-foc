# metodo-hungaro-foc
Proyecto de Fundamentos de Optimización Computacional

Ejecución de la Página Web "Método Húngaro"

Para ejecutar el Algoritmo de "Método Húngaro" se deberan seguir los pasos que a continuación se describen:

Paso 1:  Entrar en  la URL ***************  

Paso 2: Ingresar los datos: 
*Por teclado (de manera directa) ->mostrado en el lado derecho de la página
*Por archivo (cargando el archivo.txt)-> mostrado en el lado izquierdo de la página.

RESTRICCIONES PARA PASO 2:
1-Ingresar datos enteros
2-La dimensión mínima de la matriz cuadrada  es de 3*3
3-La dimensión máxima de la matriz cuadrada es de 20*20
4- Una vez que se ingresan los datos sea por teclado o por archivo, la página almacena en la BD esta información, y si se desea ingresar por teclado nuevamente otra matriz con N dimensión antes precargada, la matriz se autocargará con la información almacenada para N dimensión, si los datos quieren ser modificados se podran sobreescribir encima de cada posición de la matriz y esto no genererá ningún inconveniente. 

Paso 3: Una vez cargado los datos en la matriz, se procesa la información con el botón:   "Resolver el Problema" (mostrado debajo de la matriz completamente cargada)

Paso 4: Luego de enviar esta información, se ejecutará el algoritmo que utiliza el método húngaro paso a paso, hasta obtener la solución optima.

Paso 5: Finalmente, se muestra la matriz original, para que no haya duda de dicha solución al momento de ubicar los elementos originales, en las posiciones con cero que se visualiza al culminar este método.
